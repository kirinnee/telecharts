# TeleCharts

Telegram Chat Analyzer

# Getting Started

## Development
To run a local dev server
```bash
$ npm run dev
```
or 
```bash
$ yarn dev
```
## Unit Test
Run unit test
```bash
$ npm run test
```
or
```bash
$ yarn test
```
